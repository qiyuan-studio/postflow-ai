# PostFlow AI - 多平台社交媒体内容管理器

> 用AI自动生成适配各平台的社交媒体内容，一键发布到抖音、小红书、X(Twitter)等平台

## 功能特性

- 🤖 **AI 内容生成** - 基于DeepSeek自动生成适配各平台风格的文案
- 📱 **多平台支持** - 小红书、抖音、X/Twitter、Reddit、知乎、公众号、微博
- 📝 **SEO博客生成** - AI自动生成搜索引擎友好的博客文章
- 📊 **数据分析** - 实时追踪各平台数据
- 📅 **发布日历** - 内容日历管理，智能选择最佳发布时间
- 🔑 **API接入** - 提供REST API供第三方集成
